## 📊 What this project does

AutoInsight is a simple web app where you can **upload a CSV file and instantly understand your data**.

Instead of writing code to analyze datasets, this tool does it for you:
- Shows basic statistics  
- Finds patterns and relationships  
- Creates charts automatically  
- Gives a quick summary of your data  

It is built to make **data analysis fast, simple, and visual**.

---

## 🧠 Why I built this

When working with datasets, the first step is always:
> “What does this data actually contain?”

This app solves that problem by giving a **quick overview without coding**.

---

## ✨ Features

- 📂 Upload CSV files or paste data  
- 📋 Preview dataset instantly  
- 📊 View statistics (rows, columns, missing values)  
- 📈 Automatic charts (histogram, boxplot, scatter plot)  
- 🔗 Correlation heatmap to find relationships  
- 📝 Simple text summary of insights  
- 💾 Download cleaned data or report  

---

## 🛠️ Tech Stack

This project is built completely in Python.

- **Streamlit** → creates the web interface  
- **Pandas** → handles and analyzes data  
- **NumPy** → numerical operations  
- **Matplotlib & Seaborn** → generate charts
